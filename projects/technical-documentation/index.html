Introduction

This guide will walk through some simple C# programs to acquaint the user with the syntax. All the information of this guide comes from Microsoft® Documentation, found here.

Main Method

The main component of a C# program is the Main method. It is the entry point of a C# application, and so, when the application is started, the Main method is the first method that is invoked. The following simple code will compile and run C# code.


                class HelloWorld
                {
                    static void Main(string[] args)
                    {
                        Console.WriteLine("Hello World");
                    }
                }

Example 2: Strings

This example introduces strings.

String objects in C# are immutable. A new string is always returned. Creating them is pretty simple.

string s1 = "A string is more ";

string s2 = "than the sum of its chars.";

s1 += s2; // s1 contains: "A string is more than the sum of its chars."
Example 3: Classes

This example introduces classes.

Classes in C# are reference types. They hold reference to the address of the object on the managed heap. Instances of classes are created using the new operator. The following code shows a simple Person class, with fields and sample declaration.


                public class Person
                {
                    public string Name { get; set; }
                    public int Age { get; set; }
                    public Person(string name, int age)
                    {
                        Name = name;
                        Age = age;
                    }
                }

                class Program
                {
                    static void Main()
                    {
                        Person p1 = new Person("Teen", 17);
                        var p2 = new Person("Elder", 71)
                    }
                }
